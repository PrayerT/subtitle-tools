# Subtitle Converter & Time Shifter

A free, no-upload web tool: convert subtitles between **SRT ⇄ WebVTT ⇄ SBV**,
shift timing to fix out-of-sync captions, strip formatting tags, and export
plain-text transcripts. Batch files with ZIP download. Everything runs in your
browser — files are never uploaded, and it works offline once loaded.

**Live:** https://prayert.github.io/subtitle-tools/

Static site: `index.html` + `subtitle-core.js` (pure-JS parsing/conversion,
zero dependencies) + `app.js` (UI). Made by BitBleep.
